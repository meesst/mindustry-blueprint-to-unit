# 蓝图机动工厂 Mod

## 功能介绍

这个 Mod 允许玩家将已保存的任意蓝图转化为可移动、可操控的悬浮飞行单位。转化后的单位会保留蓝图中所有建筑的功能，包括生产、防御、电力等系统。

## 主要特性

- **蓝图转化**：将任意蓝图转化为机动单位
- **功能保留**：保留蓝图中所有建筑的完整功能
- **独立系统**：每个单位拥有独立的电力、物流和液体系统
- **物品交互**：单位可以与虚拟网格内的建筑进行物品交换
- **逻辑控制**：支持通过逻辑方块控制单位行为
- **性能优化**：实现了单位数量限制和动态更新机制，确保游戏流畅运行
- **多语言支持**：支持中英文语言

## 安装说明

1. 下载 Mod 文件
2. 将 Mod 文件放入 Mindustry 游戏目录下的 `mods` 文件夹中
3. 启动游戏，在 Mod 列表中启用 "蓝图机动工厂"

## 使用方法

1. 在游戏中放置 "蓝图装配站" 建筑
2. 点击建筑打开蓝图选择界面
3. 选择一个已保存的蓝图
4. 确保有足够的材料
5. 等待生产完成，单位会在建筑周围生成
6. 控制单位移动和执行任务

## 注意事项

- 单位数量上限为 20 个，超过后无法生产新单位
- 单位大小受蓝图尺寸限制，过大的蓝图可能影响性能
- 单位移动时，虚拟网格内的建筑会正常工作
- 单位被摧毁时，蓝图内的所有建筑也会被销毁

## 技术说明

### 核心组件

- **VirtualTileGrid**：虚拟 Tile 网格系统，用于承载蓝图中的建筑
- **SchematicParser**：蓝图解析器，用于计算单位属性和材料需求
- **BlueprintUnit**：蓝图机动单位的核心类
- **BlueprintAssembler**：生产蓝图单位的建筑
- **PerformanceOptimizer**：性能优化器，确保游戏流畅运行

### 性能优化

- 单位数量限制：最多 20 个蓝图单位
- 动态更新：优先更新距离玩家近的单位
- 关键方块优先：只更新生产、防御等关键方块
- 视距优化：只渲染可见范围内的虚拟建筑
